
# Vue.js + Vite

Огляд
--
Невеликий шаблон SPA на Vue 3 з Vite, Vuetify та управлінням станом (store). Проєкт демонструє роботу з компонентами для відображення фотографій, формою і діалогами, маршрутизацією та запитами через `axios`.

Технології
--
- Vue 3
- Vite
- Vue Router
- Vuetify
- Pinia (у залежностях) / локальний store у `src/stores/photoStore.js`
- Axios

Швидкий старт
--
Передумови:
- Node.js (рекомендовано >= 16)
- npm або yarn

Інструкції:

```bash
# Встановити залежності
npm install

# Розробка (dev сервер)
npm run dev

# Збірка для продакшна
npm run build

# Превʼю зібраного додатку
npm run preview

# Лінтинг
npm run lint

# Форматування
npm run format
```
